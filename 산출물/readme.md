FAQ  
Q1. "투표 안 한 사람(기권율)"은 단순히 '100% - 투표율'로 계산했나요?
A: 아닙니다. 투표율의 여집합을 그대로 사용할 경우 투표소에서 도장을 잘못 찍은 '무효표'까지 기권자로 섞이게 됩니다. 저희는 선관위 로우 데이터에서 투표장에 아예 나타나지 않은 '순수 기권자 수'만 추출하여 타겟 변수(Target)를 정밀하게 산출했습니다.

Q2. 과거 선거(2018년) 데이터와 최근 인구 데이터를 병합할 때 에러는 없었나요?  
A: 행정구역 개편과 표기 방식 차이로 인한 결측치(NaN) 문제가 있었습니다. 예를 들어 과거의 '강원도'는 현재 '강원특별자치도'로 바뀌었고, 기관마다 띄어쓰기 기준(예: 고양시덕양구 vs 고양시 덕양구)이 달랐습니다. 이를 해결하기 위해 자체적인 '행정구역 정규화 매핑 규칙(Mapping Rule)'을 코드에 구현하여 250개 시군구 데이터가 완벽하게 맞물리도록 병합을 완료했습니다.

Q3. 연도별로 기준이 다른 설문조사 결과는 모델에 어떻게 학습시켰나요?  
A: 8회 지선은 1~5점 척도였고, 9회 예측용 데이터는 긍정 응답 비율(%)로 조사 방식이 달랐습니다. AI 모델이 스케일 차이로 혼동하는 것을 막기 위해, 백분율 데이터를 리커트(Likert) 척도로 환산하는 선형 변환 함수를 파이프라인에 적용하여 두 데이터의 기준점을 완벽하게 동기화했습니다.

Q4. 데이터에 결측치(빈칸)가 발생한 지역은 어떻게 처리했나요?  
A: 무의미한 0으로 채우지 않고 계층적 대체(Imputation) 기법을 사용했습니다. 결측률이 30%를 초과하는 불량 피처는 과감히 제거하고, 살릴 수 있는 데이터는 '해당 지역이 속한 광역 시/도 평균'으로 1차 복원하여 지역적 특성을 최대한 보존했습니다.

Q5. 선거 예측에서 당선자가 아닌 '기권율(투표 포기)'을 예측 타겟으로 삼은 이유는 무엇인가요?  
A: 지방선거에서 판세를 뒤집는 가장 치명적인 변수는 "투표장을 아예 나가지 않는 유권자들"입니다. 양대 정당 모두에게 실망하여 투표를 포기하는 '숨어있는 침묵의 민심'의 규모를 파악하는 것이야말로, 현재 정치 트렌드와 유권자의 진짜 피로도를 읽어내는 가장 확실한 지표라고 판단했습니다.

Q6. 분석 모델에서 '나이(연령대)' 데이터를 유독 중요하게 다루는 이유가 있나요?  
A: 지방선거는 대선과 달리 세대별 투표율 편차가 매우 극심합니다. 특히 청년(2030) 층은 정치 효능감에 따라 투표 참여율이 크게 요동칩니다. 따라서 동네별로 청년과 노년층의 인구 구조 비율을 핵심 변수로 설정하여, 지역별 기권율의 베이스라인을 정교하게 예측하도록 설계했습니다.

Q7. 거대 양당을 제외한 '제3지대(소수 정당)' 지지율을 굳이 피처(Feature)로 추가한 이유는 무엇인가요?  
A: 유권자의 '진짜 속마음'을 캐내기 위해서입니다. 시장이나 구청장 투표에서는 사표 방지를 위해 양당을 찍지만, 정당 투표(비례대표)에서는 자신의 진짜 선호도를 드러냅니다. 즉, 비례대표 선거에서 제3지대 득표가 높다는 것은 양당에 실망한 유권자가 많다는 뜻이며, 이 실망감이 곧 기권으로 이어지는 핵심 인과관계를 모델에 학습시켰습니다.

Q8. 2022년(8회 지선) 모델을 가지고 2026년(9회 지선) 결과를 예측할 때, 어떤 점을 주의했나요?  
A: 2022년 지선은 대선 직후에 치러진 이른바 '허니문 선거'였지만, 2026년은 유권자들의 피로도와 정치적 지형이 완전히 다릅니다. 따라서 과거의 투표율 숫자에만 의존하지 않고, 최신 유권자 의식조사(정치 무관심도, 공정성 인식 등)를 미래 예측 파이프라인의 핵심 가중치로 반영하여 변화된 민심을 잡아냈습니다.
